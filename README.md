# Pharmacy Inventory Management System

A desktop-based pharmacy inventory management system developed using
Java Swing and JDBC.

## Features
- Medicine management (Add / Update / Delete)
- Inventory and stock control
- Billing system
- User-friendly Swing GUI
- MySQL database integration

## Tech Stack
- Java
- Swing
- JDBC
- MySQL

## Database Setup
1. Install MySQL
2. Create database:
   CREATE DATABASE pharmacymanagementsytem;
3. Import database:
   mysql -u root -p pharmacymanagementsytem < pharmacymanagementsytem.sql

## Run Application
java -jar PharmacyInventory.jar
